Quarkus

  • Conceitos

    • Quarkus é uma estrutura Java nativa do Kubernetes introduzida pela Red Hat em 2019. É construída sobre padrões como MicroProfile, Vert.x, Netty e Hibernate.
    • O objetivo do Quarkus é tornar o Java uma plataforma líder no Kubernetes, permitindo uma inicialização mais rápida, baixo consumo de memória e escalonamento quase instantâneo em plataformas de orquestração de contêineres. O Quarkus consegue isso usando plug-ins Maven personalizados para realizar o máximo de trabalho possível durante o tempo de compilação em vez do tempo de construção (no Quarkus, isso também é chamado de inicialização em tempo de compilação ).
    • O Quarkus está usando principalmente tecnologias padrão existentes, mas está aberto para extensão. No entanto, como o projeto foi iniciado há apenas um ano, a maturidade e compatibilidade dessas extensões nem sempre são claras. É provável que isso mude no futuro, conforme a plataforma crescer.
