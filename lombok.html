Lombok

Lombok é um plug-in de compilador

  • Conceitos

    • Lombok disõe de uma série de recursos (anotações) que possibilitam a codificação mais elegante e limpa no java
    • Uma boa utilidade para o lombok é a redução de código em classes DAO
    • Repositorio com exemplos lombok: Lombok GitHub
    • Repositório com Wiki de annotations lombok: Lombok Wiki Annotations
    • Lombok permite anotações que facilitem administrar logs
    • A utilização do recurso de-lombok permimte ao desenvolvedor visualizar todo o código nativo gerado pelas anotações do lombok
    • Impacto do lombok nos projetos java, a redução exorbitante da quantidade de código nátivo padronizado
    • A tão solicita utilização da anotação @NonNull possíbilitada por este recurso
